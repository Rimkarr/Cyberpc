# 🖥️ CyberPC Kazakhstan v2.0

Профессиональный интернет-магазин и конфигуратор ПК.

## 🚀 Быстрый запуск

```bash

npm install

# 2. Запустить сервер
npm start

# 3. Открыть в браузере
http://localhost:3000
# или
http://127.0.0.1:3000
```

## ✅ Функции сайта

### 🛍️ Магазин
- Каталог с 60+ товаров (CPU, GPU, RAM, SSD, PSU, Case, Cooler, Motherboard)
- Фильтры: категория, бренд, цена (мин/макс), сортировка
- Вид сетка / список
- Быстрый поиск в шапке (🔍)
- Теги товаров: Хит, Топ, Бюджет, Gaming, 4K, Pro

### 🔧 Конфигуратор ПК
- 8 слотов: CPU, Материнская плата, GPU, RAM, SSD, Кулер, БП, Корпус
- **Проверка совместимости в реальном времени:**
  - Сокет CPU ↔ Материнская плата (AM4, AM5, LGA1700...)
  - Тип памяти DDR4/DDR5 ↔ Материнская плата
  - Мощность БП ↔ TDP видеокарты + процессора (с 25% запасом)
  - Рекомендация минимального БП
- Несовместимые компоненты затемнены в модальном окне
- Сохранение сборки (для авторизованных)
- Добавить всю сборку в корзину

### 📦 Корзина и заказы
- Добавить / удалить товар
- Изменение количества (+ / −)
- Оформление заказа: имя, телефон, адрес, комментарий

## 🗄️ База данных

Используется **NeDB** — встроенная NoSQL база (работает без установки MongoDB/PostgreSQL).

Данные хранятся в папке `data/` в файлах `.db`:

| Файл | Что хранит |
|------|-----------|
| `users.db` | Пользователи: логин, email, пароль (bcrypt хеш), роль |
| `products.db` | Товары: все характеристики, история цен, количество просмотров |
| `orders.db` | Заказы: состав корзины, контакты покупателя, статус |
| `builds.db` | Сборки: пресеты и пользовательские конфигурации |
| `chat.db` | История чата с ботом |
| `reviews.db` | Отзывы на товары |

**Где смотреть данные?** Файлы `.db` — это обычный текст (JSON), можно открыть в любом редакторе.

## 🔒 Безопасность

- Пароли хешируются через **bcrypt** (нельзя восстановить)
- Сессии через **express-session** (хранятся в памяти сервера)
- Загрузка файлов только для admin (через multer, только изображения, макс 5MB)

## 📸 Как добавить изображения товарам

**Вариант 1 — через URL:**
1. Найдите изображение товара в интернете
2. Скопируйте прямую ссылку на картинку
3. Вставьте в поле "URL изображения" при создании/редактировании

**Вариант 2 — загрузить файл:**
1. Войдите как admin
2. Откройте карточку товара → кнопка "📁 Загрузить фото"
3. Или в Админ-панели → Изменить → поле "Загрузить новое изображение"

Файлы сохраняются в `public/uploads/`
